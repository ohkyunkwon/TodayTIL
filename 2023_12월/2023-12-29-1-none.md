
## Gradle이란 ?
* Groovy 기반의 오픈소스 빌드 도구
## Groovy 기반 이란?
* JVM에서 실행되는 스크립트 
* 언어문법이 Java와 아주 가깝다
* Java와 호환이 되며 Java 클래스 파일을 그대로 Groovy 클래스로 사용할 수 있다

## 빌드 도구란?
* 소스 코드를 실행 가능한 어플리케이션으로 만들어주는 도구

## Gradle 장점 
* Gradle은 Groovy 문법으로 간결한 스크립트를 작성할 수 있습니다
* 빌드 속도가 빠르다
* Gradle은 멀티 프로젝트 빌드 기능을 제공하기 때문에 이런 번거로움과 실수를 줄일 수 있다
